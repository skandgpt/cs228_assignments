According to the syllabus, the exam weighs 17% of your final grade. The material will be primarily based on the lecture notes. Though students are responsible for taking notes in class, the instructors have been posting class notes and materials.

The scope of the exam includes the following topics:

  • Generic types & methods
    • Type parameters and instantiations
    • The Comparable and Comparable interfaces
    • Generic sorting
    • Generic classes vs. generic methods
    • Scope of type variables
    • Wildcards: lower and upper bounds on a type
    • Erasure
  • Collections and iterators
    • Methods of the Collection interface
    • Methods of the Iterator interface
    • Usage of collections and iterators
    • The AbstractCollection class
    • The MyFirstCollection class
      • Big-O operation times
      • Iterators
  • Linked lists
    • Doubly-linked collections
      • The DoublyLinkedCollection class
      • The Node inner class
      • Iterators: cursor, hasNext(), next(), and remove()
      • Pointer updates
      • Removal (general and special cases)
      • Big-O run times of list operations
  • The List interface
    • Access methods: add(), get(), set(), and remove()
    • The ListIterator interface
      • The logical cursor
      • nextIndex() and previousIndex()
      • Methods: hasNext(), next(), hasPrevious(), and previous()
      • Mutators: remove(), add(), set()
  • The DoublyLinkedList class
    • Private helper methods: link(), unlink(), and findNodeByIndex()
    • The two add methods
    • List iterators
      • Instance variable: cursor
      • Two constructors
      • add()
      • hasNext(), next(), hasPrevious, and previous()
    • Big-O times of list operations
  • Stacks
    • Stack underflow and overflow
    • Array-based stack
    • Matching parentheses
  • Postfix and infix expressions
    • Conversions between infix and postfix (by hand)
    • Postfix evaluation using a stack
    • Rank of an infix expression
    • Precedence and associativity
    • Infix-to-postfix conversion using a stack: input and stack precedences
    • Possible errors
    • Running times of postfix evaluation and infix-to-postfix conversion
